# GitHub Hosts

This Java-based tool can help you generate a host file to access GitHub more quickly.

## How to Use

1. Copy the hosts content above
2. Add it to your system's hosts file:
   - Windows: `C:\Windows\System32\drivers\etc\hosts`
   - Linux/Mac: `/etc/hosts`
